# Phoenix

Phoenix is a C compiler written in C.

## Scanner "Lexi"

Lexical analysis of ASCII files.

## Parser

Semantical analysis LL(1) style.

## Target Machine "Margit"

Emulates a RISC (DLX) system with 32bit words.
